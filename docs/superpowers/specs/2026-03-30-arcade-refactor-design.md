# Kinetic Arcade 前端重构设计文档

## 概述

基于 Figma 设计图对 Kinetic Arcade（霓虹贪吃蛇）前端进行全面重构，采用 Neon 霓虹视觉风格。

**Figma 设计源**: https://www.figma.com/design/DE9JjeeIFpR2Rc4eixrkAx/home

---

## 视觉风格规范

### 色彩系统
| 用途 | 色值 |
|------|------|
| 主背景渐变起点 | #1a1a2e |
| 主背景渐变终点 | #16213e |
| 霓虹绿强调 | #4ade80 |
| 蓝色辅助/按钮 | #409eff |
| 卡片背景 | rgba(26, 26, 46, 0.9) |
| 卡片边框 | #2a2a4e |
| 文字主色 | #ffffff |
| 文字次要 | #909399 |
| 文字禁用 | #666666 |

### 组件样式
- **卡片圆角**: 16px
- **按钮圆角**: 8px
- **输入框圆角**: 8px
- **卡片阴影**: 使用 CSS box-shadow 模拟发光效果
- **装饰元素**: 圆角矩形 + backdrop-filter: blur()

---

## 页面重构详情

### 1. 登录页 (LoginView)

**布局**: 左右分栏 (60% / 40%)

**左侧品牌区**:
- 顶部标签: "KINETIC ARCADE" (12px, 霓虹绿)
- 主标题: "霓虹贪吃蛇" (48px, 白色, 渐变或发光效果)
- 欢迎语: "欢迎回到充满活力的竞技场..." (16px, 次要色)
- 装饰头像: 3个小头像堆叠 + "+N" 文字

**右侧登录卡片**:
- 卡片宽度: 400px
- 内边距: 32px
- 标题: "登录账号" (18px)
- 表单字段:
  - 用户名或邮箱 (带左侧图标)
  - 密码 (带左侧图标 + 右侧显示/隐藏切换)
- 选项行: "记住我" 复选框 + "忘记密码？" 链接
- 主按钮: "登录" (霓虹绿背景, 黑色文字)
- 分隔线: "或"
- 社交按钮: "Google 登录" (灰色背景)
- 底部链接: "还没有账号？立即注册"

**装饰元素**:
- 左上角: 大圆形模糊装饰 (512x512)
- 右下角: 圆形模糊装饰 (384x307)
- 右上角: 圆角方形边框装饰

---

### 2. 注册页 (RegisterView)

**布局**: 居中单卡片

**页面结构**:
- 顶部导航: Logo (左) + 关闭按钮 (右)
- 居中注册卡片 (480px 宽):
  - 标题: "注册新账号"
  - 副标题: "加入 Kinetic Arcade，体验次世代贪吃蛇竞技。"
  - 表单字段 (从上到下):
    1. 邮箱输入 (带图标)
    2. 验证码输入 (左侧输入 + 右侧"发送验证码"按钮)
    3. 密码输入 (占左半)
    4. 确认密码输入 (占右半)
  - 主按钮: "注册" (霓虹绿)
  - 分隔 + 社交登录
  - 底部链接: "已有账号？立即登录"

**装饰元素**:
- 左上角: 大圆形模糊 (384x384)
- 右下角: 圆形模糊 (500x500)
- 右上角: 浮动信息卡片 (200x140) - 包含图标+文字
- 左下角: 玩家头像组 (3个头像堆叠)

---

### 3. 游戏页 (GameView)

**布局**: 左游戏区 + 右侧边栏

**左侧游戏区** (816px 宽):
- 外层容器: 816x896, 圆角渐变背景
- 内层画布容器: 752x752, 居中
  - 游戏画布: 居中显示
  - 覆盖层: 开始按钮 + "开始游戏" 文字
- 预游戏状态: 显示速度选择 + 开始按钮

**右侧侧边栏** (320px 宽):

**得分展示区**:
- 标签: "当前得分" (小字)
- 得分数字: "1,240" (大号数字, 霓虹绿)
- 分隔线
- 本局信息:
  - 速度倍数 (标签 + 数值)
  - 得分倍数 (标签 + 数值)

**排行榜入口按钮**:
- 图标 + "查看排行榜" 文字
- 右侧箭头

**Bento 网格布局** (速度/得分倍数):
- 左卡片: 速度图标 + "速度倍数" + "1.25x"
- 右卡片: 得分图标 + "得分倍数" + "3x"

**玩家成就预览**:
- 左侧图标
- 右侧: 成就名称 + 进度

**键盘控制区**:
- 4个方向键可视化 (上右下左)
- 空格暂停按钮

**顶部导航栏**:
- Logo: "霓虹贪吃蛇"
- 右侧: 静音按钮 + 设置按钮 + 用户头像

---

### 4. 排行榜弹窗 (LeaderboardModal)

**布局**: 居中 Modal

**尺寸**: 672px 宽, 高度自适应

**结构**:
- **Header**: "排行榜" 标题 + "close" 关闭按钮
- **Tab 行**: 全部 / 我的排名
- **表格**:
  - 表头: 排名 | 玩家 | 分数 | 用时
  - 第1名: 金色高亮背景, 皇冠图标
  - 第2名: 银色背景, 奖杯图标
  - 第3名: 铜色背景, 奖杯图标
  - 其他行: 交替背景色
- **Footer**: "收起" 按钮

## 组件清单

### 基础组件
| 组件 | 说明 |
|------|------|
| NeonCard | 霓虹风格卡片容器 |
| NeonButton | 霓虹风格按钮 (primary/default) |
| NeonInput | 霓虹风格输入框 (带图标支持) |
| NeonCheckbox | 霓虹风格复选框 |
| NeonModal | 霓虹风格弹窗 |
| GradientBackground | 渐变背景容器 |
| DecorativeCircle | 模糊圆形装饰 |
| SocialLoginButton | 社交登录按钮 |

### 业务组件
| 组件 | 说明 |
|------|------|
| GameBoard | 游戏画布容器 |
| ScoreDisplay | 得分展示组件 |
| SpeedMultiplier | 速度倍数显示 |
| ScoreMultiplier | 得分倍数显示 |
| LeaderboardTable | 排行榜表格 |
| LeaderboardRow | 排行榜行 |
| TopNavBar | 顶部导航栏 |
| BottomNavBar | 底部导航栏 |
| DirectionalKeys | 方向键可视化 |

## 优先级排序

1. **Phase 1**: 登录页重构
2. **Phase 2**: 注册页重构
3. **Phase 3**: 游戏页重构
4. **Phase 4**: 排行榜弹窗重构

---

## 技术约束

- **框架**: Vue 3 + Composition API
- **UI 库**: Naive UI (暗色主题)
- **状态管理**: Pinia
- **路由**: Vue Router
- **构建工具**: Vite
